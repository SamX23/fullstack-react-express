## Cara penggunaan

Membutuhkan `npm` atau `yarn` terinstall di PC/laptop.

- Clone repository, lalu buka pada editor.
- Jalankan `npm run client-server` atau `yarn client-server` untuk install library.

### Menjalankan Client

Jika hanya develop client, cukup jalankan ini

- Jalankan `npm run dev-client` atau `yarn dev-server` untuk menjalankan client.

### Menjalankan Server atau client-server side

Jika hanya develop server, cukup jalankan `dev-server`, jika ada halaman yang ingin diakses dari client, silahkan jalakan `build` lebih dulu, berikut tahapannya.

- Jalankan `npm run build` atau `yarn build` untuk build frontend agar bisa diakses server.
- Jalankan `npm run dev-server` atau `yarn dev-server` untuk menjalankan server.
